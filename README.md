Hi, I'm Daniel Púa (a.k.a. devploit), offensive security researcher based in Málaga (Spain). I do bug bounty, penetration testing, and security consulting — focused on web, mobile, APIs, and source code. Currently Head of Security at [Magnific](https://www.magnific.com/) and CTF competitor with [SFPE](https://ctftime.org/team/227086). I was previously part of [ripp3rs](https://ctftime.org/team/50984) and represented Spain in the European Cybersecurity Challenge (ECSC) as a member of the national team.

<img align="right" alt="GIF" src="https://s6.gifyu.com/images/bpNKb.gif" width="350" height="200" />

I maintain [pwny.cc](https://www.pwny.cc) — a curated repository of offensive payloads and research material for the security community.

In my spare time, I build open source security tools — always with the same mindset: _If it parses, it breaks._ 🧨
